# Driving School Website

A modern and responsive landing page built for a driving instructor.

## Features

- Responsive design
- Modern UI/UX
- FAQ section
- Student testimonials
- WhatsApp contact integration
- TikTok video integration
- Lead generation form
- Mobile-friendly layout

## Technologies Used

- HTML5
- CSS3
- Font Awesome
- Google Fonts

## Purpose

This project was created as a promotional website to help a driving instructor showcase services, increase trust, and generate new student leads.
